## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/AppRonin/company-scan.git
cd company-scan
```

### 2. Install dependencies

```bash
cd frontend
flutter pub get
```

### 3. Run the app

```bash
flutter run
```
